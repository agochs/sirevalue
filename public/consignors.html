How this works. Every realized pinhook pair (same horse, yearling sale → 2YO sale) is bucketed under both the yearling consignor and the 2YO consignor. Consignors with fewer than 3 pairs are excluded as too noisy. Names are lightly normalized (", Agent", ", for X" suffixes stripped) so the same firm doesn't appear under multiple variants. Sample sizes vary widely; default filter is 10+ pairs.
