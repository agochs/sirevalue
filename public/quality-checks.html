Quality Checks

Pairs flagged for verification.

Cohort fingerprinting is biologically robust — mares only foal once a year, so a (sire, dam, foal year) tuple should be unique. But upstream data can have dam-name typos, transposed prices, or genuinely extreme outliers. This page surfaces every matched pair that tripped a heuristic check, so we can verify (or correct) before quoting them publicly.

Every flagged pair is still counted in the headline pinhook stats — these are flags, not exclusions. If you spot a confirmed bad match, drop it from the source data and rerun.

Flag definitions.
extreme gain
Realized return ≥ 1,500%. Genuine but rare; verify the dam's name in both sale catalogs matches.
extreme loss
Realized return ≤ −75%. Often legitimate (injury, soundness issue), but verify the 2YO sold for what we think.
micro yearling → big 2YO
Yearling sold for under $5K but 2YO fetched ≥ $100K. Most common false-match shape if a dam-name typo elsewhere created a phantom cohort.
multi-yearling/multi-2YO
Multiple yearling and 2YO entries share the same cohort fingerprint. Biologically impossible without a dam-name collision.
sub-$1K price
Either yearling or 2YO sold for under $1K. Likely transcription noise or unusual private terms.
