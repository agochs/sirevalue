Each hip on the catalogs page gets a predicted signal tier (strong / positive / neutral / weak / none) based on the sire's realized pinhook track record. This page checks the prediction: when the hammer fell, did strong-signal hips actually sell for more, and did they generate higher realized pinhook returns?

How to read this. Each row is one signal tier. Median sold and Mean sold are the actual hammer prices for hips of that tier. Realized return only fills in when the same horse's yearling sale price is on file (matched by sire+dam+foal year) — that's a true pinhook outcome. A monotonic decrease from strong → weak (green up-arrow) indicates the signal is doing real work; a flat or inverted column means that metric isn't sensitive to pinhook history at this slice.
